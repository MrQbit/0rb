Install

Prerequisites: Node.js 20+ and the GitHub CLI (run gh auth login first).
gh release download -R MrQbit/rak00n -p "rak00n-installer-macos" -D /tmp --clobber && chmod +x /tmp/rak00n-installer-macos && /tmp/rak00n-installer-macos && rm /tmp/rak00n-installer-macos && source ~/.zshrc
gh release download -R MrQbit/rak00n -p "rak00n-installer-windows.exe" -D $env:TEMP --clobber; & "$env:TEMP\rak00n-installer-windows.exe"

The installer downloads the latest rak00n binary, installs it to ~/.rak00n, and adds it to your PATH. If no API key is configured, rak00n will automatically set up a local Ollama instance.

What It Does

Zero-Config Setup Installs Ollama and pulls a model automatically if no API key is configured.
200+ Models Works with OpenAI, Gemini, Azure, DeepSeek, Ollama, OpenRouter, and more.
AI Coding Agent Reads, writes, and runs code in your terminal. Understands your full codebase.
Cross-Platform macOS, Windows, and Linux. One command to install, one command to run.

After Install

# Open a new terminal, then: rak00n

rak00n will detect your environment and guide you through provider selection. To use your own API key, set OPENAI_API_KEY or ANTHROPIC_API_KEY before running.
